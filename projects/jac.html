JAC

JAC – Gamer Matchmaking Web App As part of a team of three, I developed JAC, a full-featured web application aimed at connecting gamers based on shared interests and playstyle preferences. The app uses a custom-built matching algorithm to recommend potential friends and community groups tailored to each user’s profile. Users can create an account, log in securely, edit their profile, add and manage friends, send real-time messages, and participate in or create gaming communities. One of the key features is a personalized recommendations page, which dynamically suggests users who match the logged-in user’s preferences. We built JAC using HTML, CSS, and JavaScript for the front end, and Firebase Firestore for real-time database management and backend services. This project helped me strengthen my full-stack development skills and collaborate effectively within a small agile team.
