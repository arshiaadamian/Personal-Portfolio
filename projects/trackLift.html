TrackLift

TrackLift is a fitness app I built to solve a personal pain point—keeping track of daily workouts. I used to write my workout plan on paper, which often got lost or damaged. I wanted a clean, structured way to follow my routines, knowing exactly which muscle groups I was targeting each day and how many sets I had for each exercise. While I found apps that offered this functionality, most required a paid subscription,so I decided to build my own.

TrackLift allows users to:

  • Create and save personalized workout plans for each day of the week
  • View their daily workout plan on a “Today” page
  • Access curated workouts for beginners, each linked to instructional YouTube videos

This project was built using JavaScript, React, Node.js, CSS/Bootstrap, and MongoDB for the back-end database.
